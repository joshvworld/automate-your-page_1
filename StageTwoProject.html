Stage 1: Webpages, Documents, and Structure

The Basics of the Web and HTML

World Wide Web

The web is the way computers interact with the internet via HTTP, or hypertext transfer protocol. The webpages we see are transmitted via this protocol and are written in HTML, hypertext markup language.

HTML

HTML documents are written using markup language, or tags, to alter plain text and present the multimedia experiences that we know as webpages. Different tags have different behaviors. Some must be accompanied by a closing tag, while others are void tags. Some tags are block tags and segregate content into invisible boxes, while others are inline and simply affect the text without blocking it off.

Stupid Computers

We say that computers are stupid because they cannot detect and fix mistakes that a reasonably intelligent human would be able to. They can only interpret instructions literally and cannot guess context.

Programming has too many rules and details to remember everything. Programmers should focus on learning big, important details and rely on their ability to search for and find the rest. Creating a Structured Document with HTML

What a Front-End Web Developer Does

The job of a front-end web developer is to translate the visual design of a website into an actual, functional site. The tools they use for these sites are HTML, CSS (Cascading Style Sheets), and JavaScript, which we'll discuss later.

HTML & CSS

HTML files provide the structure of the site. HTML is the language of the web, with syntax and rules, composed primarily of tags that instruct the browser how to display the content. CSS files determine the style of the site. Webpages with identical HTML may look radically different depending on their CSS file. CSS may instruct the browser how to display fonts and colors, as well as shapes and sizes of objects.

The DOM Tree

The browser renders HTML documents into a tree-like structure called the DOM, or Document Object Model, tree, that we can see in the developer tools of a browser. Using this tree, we can peruse through each element in a logically-organized manner.

Boxifying (A Real Word)

Despite appearances, all objects on a webpage are contained in rectangular boxes which may or may not be visible on the page. A designer presents the front-end web developer with a design for the website, who then breaks down the page into basic component boxes from big to small. These boxes are then defined with div tags, which are accompanied by class names to instruct the CSS file how to treat each box. You can see what a box looks like by clicking here.

Creating a Structured Document with HTML

How to Use CSS

With no CSS, browser uses its default style to display elements, resulting in a very basic look. CSS can either be placed in the head of the HTML file or placed in a separate file. This may help you stay organized on larger projects. Another method which has the high potential for unnecessary repetition and makes HTML sites more difficult to maintain or update is the inline use of the <style> tag for each element.

Composition of the Box

The boxes that make up webpages are composed of the margin, border, padding, and content. Content is the image or text, a sort of nucleus surrounded by other sections. Padding clears an area around the content within the border. The padding is affected by background color of box. The border goes around padding and content. The border is inherited from the color property of the box. The margin, or space between boxes, clears an area around the border. It is completely transparent and colorless.

Box Sizing

The element size is equal to the sum of the content, padding, and border widths. In order to maintain uniform box size without recalculating each time you change one of those values, use the box-sizing: border-box; CSS style. The size of the box can be set in pixels or percentage. Setting a box size as a percentage will cause the box to resize based on display size or size of browser window. Setting a box size in pixels will cause box to always display in exactly the same size, which may lead to less than desirable results on small displays or when the browser window is resized smaller. You can also set a max width for the box. In that case, box will resize and become smaller than necessary but never larger than the max width.

Box Positioning

Positioning boxes on the page is one of the most complicated parts of working with CSS. Divs are block elements (as opposed to inline), so by default they take up the entire width of a page. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.

Avoiding Repetition: How to Not Repeat Yourself or Do the Same Thing Over and Over. This Should Be Avoided and Also You Shouldn't Do It.

It cannot be stressed enough how important it is to avoid repetition in your code. This not only makes it so that you don't have to repeat yourself over and over, but also helps you to avoid errors that might come from having to change so many individual elements. This allows you to set CSS selectors such as class and ID and change all elements with those selectors using one CSS command.

Recap and Recipe for Successful Web Developing

  1. Look for natural boxes in a design layout.
  2. Look for repeated styles and semantic elements.
  3. Write your HTML using appropriate tags and selectors.
  4. Apply styles (from BIGGEST to smallest).
  5. Fix things.
    • Resize the browser.
    • Try out your code in different browsers.

Stage 2: Telling Computers What to Do

Introduction to "Serious" Programming

What is Computer Science?

Computer Science is how to solve problems like building a search engine by breaking them into smaller pieces and then precisely and mechanically describing a sequence of steps to solve each piece, which can then be executed by a computer. For example, a search engine is built by building three main pieces: finding data by crawling webpages, building an index to respond quickly to queries, and ranking pages to give the best result for a given query.

Programming

A computer is a universal machine which can be programmed to do essentially any computation. Computers need programs, however, to carry out these computations. Without a program, however, the computer can do nothing. A program is simply instructions telling the computer what to do. The program needs to be a very precise sequence of steps since computers can only do what they're told (for now). The computer can execute these steps very quickly.

Python

Python is an interpreter that runs our programs, interprets them, executes the program we wrote in the Python program by translating them into instructions the computer understands. There are two parts to the Python interpreter: the environment where you input your code and the output panel where the result of your code is displayed.

Why Programming Languages?

We use computer programming languages such as Python to avoid the ambiguities present in our natural languages such as English or Mandarin. An example of such an ambiguity is the word biweekly, which can mean once every two weeks or twice a week. Furthermore, natural language is too verbose to write efficient code.

Decimals and Integers in Python

When the input values for a mathematic equation are integers (i.e., they do not contain a decimal), python will shave any decimal value off of the result. For example, the expression 5 / 2 will yield a result of 2 instead of 2.5 and 1 / 10 will yield a result of 0 instead of 0.1. Adding a decimal to one of the values will force Python to give an answer that includes a decimal and the precise answer.

Variables and Strings

Variables

In order to reduce confusion over what each value represents, it is useful to assign variables for values in calculations. We use an assignment statement to assign these variables, which begin with either a letter or an underscore. An example assignment statement is speed_of_light = 299792458 which represents the speed of light in meters per second. After assigning the variable, can use this variable in future equations rather than remembering the exact number. If, for example, we wish to calculate the speed of light in centimeters, we would type print speed_of_light * 100 into Python since our speed of light is the distance light travels in one second in meters and there are 100 centimeters in a meter.

Variables Can Vary

It is important to note that because variables can be superseded by other identically-named variables that follow later in the code. Therefore, it is also important to remember that the symbol = does not mean equals in computer programming as it does in mathematics. It simply means assignment, which is to say that whatever value follows the symbol = is assigned to the variable that precedes the symbol.

Strings

A string is a sequence of characters surrounded by either single or double quotes. It is imperative that if we begin a string with a single quote, we must also end it with a single quote and the same is true of double quotes. If using single quotes to define the string, then double quotes may be used inside the string and vice versa. For example, 'Hello', 'He"llo' and "Hello" are all valid strings, while 'Hello", "Hello', and 'He'llo' are not. Strings and numbers cannot be combined. The command print 'Hello' + 1 would result in an error. However, strings can be combined using the + symbol which concatenates strings together. For example, the command print 'Hello' + 'Josh' would concatenate the two strings, producing the result HelloJosh. Note that Python does not automatically add a space when concatenating two strings.

Indexing Strings

We can extract subsequences from strings. We cannot do this with numbers. Characters in a string are numbered beginning with 0. For example, in the string 'Josh', the number 0 would correspond with J, 1 would correspond with o, et cetera. To extract part of the string, you would use square brackets [ ] to surround an expression that evaluates to a number representing a character in the string. In the example of print 'Josh'[2], the result would be s because that is the character in position 2 of the string. However, print 'Josh'[4] would result in an error because there is no character in position 4 of the string.

Selecting Subsequences Greater than One Digit Long

In order to extract a subsequence greater than one digit, we use a colon surrounded with numbers or expressions that resolve to numbers on either side of the colon enclosed in square brackets that represent the starting point and ending limit of the subsequence. The result will include the positions from the number on the left side of the colon up to, but not including, the number on the right side of the colon. If either side of the colon is left blank, the result will include all characters from the beginning or until the end of the string. For example, the expression print Josh'[1:] will yield the result osh.

Finding Strings in Strings

Find is a function built into Python which allows us to search for strings inside of larger strings. The format for using the find function is print 'string'.find('string') and the result will be the number where the string being sought begins in the original string. For example, print 'Josh'.find('o') will yield a result of 1 since the o first occurs in position 2 of the string 'Josh'. If the search string does not occur in the original string, then the result will be -1, as will occur with the command print 'Josh'.find('a'). We can also use additional parameters in the find function by separating the string being searched for with a comma and a number indicating the position to begin the search from after the comma. For example, the command first_string.find('string',5) will search for the string 'string' starting from position 5 in the string first_string.

Input -> Function -> Output

What is a Function?

Functions are a way of taking inputs to produce outputs in coding. This is a powerful tool that avoids the necessity of producing code for each individual desired result. For example, if we want to calculate squares of numbers without typing in print x * x, print y * y, print z *z, etc. each time, we can simply define a function that will allow us to achieve the desired result like
def square(n):
    return n * n

print square(5)
for which the result would be 25. To produce any further squares, we would only need to replace 5 with any number of our choosing. Functions can also be used to concatenate strings. For example, the function
def abbaoncrack(a, b):
    return a + b + b + a + a + b + b + a + a + b + b + a

print abbaoncrack('Mamma', 'Mia')
would produce the result MammaMiaMiaMammaMammaMiaMiaMammaMammaMiaMiaMamma.

Inputs and Outputs

Functions take input, do something with it, and then produce output. The command def defines the input and the command return defines what to do with the input to create the output. Note that when defining the function, there can be any number of inputs within the parentheses. Furthermore, without the return, no output is produced.

Telling Computers What to Do

Equality Comparisons

We can make code behave differently based on decisions. One way to do this is by making comparisons. In order to make comparisons, we use the mathematical symbols <, >, <=, >=, !=, and ==. For equal, we use a double == instead of a single = because as we learned before, a single = means assignment where we assign a value to a variable. The result of these comparisons is either of the boolean values True or False.

If Statements

We can use comparisons to make our code different based on the result of a comparison. We can define an output to occur based on whether or not the input meets certain criteria. We write this as
if <TestExpression>: 
    <block>
so that if the text expression evaluates to true, then the block is executed. If it evaluates to false, then the block is not executed and the next line of code is processed.

Or

We can also use the or construct to evaluate comparisons. A unique thing about using or is that it only evaluates as much as it has to. This means that in the code <Expression> or <Expression>, if the first expression evaluates to True, then the value is True and the second expression is not evaluated. If the value of the first expression values to false, then the value of the construct is the value of the second expression.

While Loops

If statements can only execute the code in the <block> either 0 or 1 times depending on if the <TestExpression> evaluates to true or not. While statements, on the other hand, continue to execute the code contained in the <block> until the <TestExpression> evaluates to false. While loops are written like
while <TestExpression>: 
    <block>

Break Statements

Using the python command break in a while loop enables us to exit the while loop mid-process if certain conditions are met. Writing break into the code we know now is not a very good idea as it is extraneous and more complicated than necessary; however, this will come in handy later when extracting all the links in a page.

Structured Data: Lists and For Loops

Lists

A string is a kind of structured data that is a sequence of characters. We can break it down into its characters and operate on subsequences of strings. Strings are limited, however, because we can only put characters in strings. Lists, on the other hand, are much more powerful because we can put a sequence of anything into a list, including strings, numbers, and other lists. These elements are separated by commas. Just as with strings, we can use square brackets to access elements in lists. The major difference is that instead of returning a character, we will return an item or items in the list. The grammar for lists is a square bracket enclosing from zero to infinitely many elements separated by commas.

Mutation

Another major difference between lists and strings is that lists are mutable. This means that we can change the value of a list after we've created it. With strings, you can use an assignment statement to create a variable. You can then use a new assignment statement to change the variable to reference a different string. This doesn't change the string, it simply creates a new string to be referenced by the variable. Additionally, you use another assignment statement to change the same variable to reference a previous iteration of itself and concatenate that with another string but that still only changes what the variable refers to and doesn't modify the initial string.

On the other hand, we can modify lists by modifying the value of their elements. For example, in the list p, we can use p[0] = x to change the value in the first position of list p to whatever x is, be it a string, number, or even another list.

Aliasing

When we have two different ways to refer to the same object, that's referred to as aliasing. We can use an assignment statement to alias a list to correspond to multiple values. For example, for list p, we can use the assignment statement q = p so that q and p both refer to the same list. Therefore, when modifying either p or q, the list that both of these variables refer to will be altered.

List Operations

  1. Append adds a new element to the end of a list. Note that it only adds whatever is being appended to the newly created last position in a list, so that if another list is appended, it is nested within only one position of the list. This is different than the following operation, plus.
  2. Plus (+) produces a new list that combines the elements of two lists into a new list. It works similarly to concatenation of strings by adding a second list to the end of a first list.
  3. Len, short for length, shows how many elements are in a list, string, or any object that is a collection of things. It only counts outer elements, so that if one of the elements of a list is a nested list, it only returns one for the entire nested list.

For Loops

Since lists are a collection of things, it's very useful to loop through a list and do something to every element in a list. We can use a new loop known as a for loop to remove the complication of using a while loop on structured data. The grammar for forming a for loop is
for <name> in <list>:
    <block>
where each element in the list is given the name <name> and the block is evaluated for each element.
